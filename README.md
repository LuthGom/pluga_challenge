# Desafio Pluga Desenvolvedor

Esse projeto foi o resultado de um desafio técnico para o processo seletivo da Empresa Pluga.

O projeto consiste em criar uma tela com as ferramentas da lista dada, com paginação de 12 em 12 conforme o wireframe. Além disso, a aplicação possui:

- Busca por nome das ferramentas.
- Quando um item é clicado, abre-se um modal apresentando a ferramenta.
- O modal mostra também uma lista com as 3 últimas ferramentas clicadas pelo usuário.

## Como executar (rápido)

1. Clone o repositório:

```powershell
git clone https://github.com/LuthGom/pluga_challenge.git
cd pluga_challenge
```

2. Instale as dependências:

```powershell
npm install
```

3. Rode a aplicação em modo de desenvolvimento:

```powershell
npm run dev
```

Abra http://localhost:3000 no navegador (ou a porta informada no terminal).

## Notas

- Projeto feito com Next.js (App Router) e React.
- Estilos podem usar Tailwind e/ou CSS Modules conforme a organização do projeto.
- Se necessário, ajuste a versão do Node.js conforme o arquivo `package.json`.
